# PaperTrail

**Project Description:**
Paper Trails is a web-based single page application that allows users to log in, explore creative artwork, submit sketch requests using images or descriptions, and manage user data through a role-based interface using a mock backend.

**Technologies Used:**

* React JS
* React Router
* JSON Server
* HTML, CSS, JavaScript
* Node.js

**How to Run the Project:**

1. Install Node.js on your system.
2. Open the project folder and run `npm install`.
3. Start JSON Server using `npx json-server --watch db.json --port 3000`.
4. In a new terminal, start the React app using `npm start`.
5. Open `http://localhost:3000` in your browser.
